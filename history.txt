### How to use it
1. Copy the JSON above into a file named **`weather_workflow.json`**.
2. In n8n, open **Import** → **Import from File** and select the `weather_workflow.json` file.
3. After importing, the workflow will appear with the four nodes linked as follows:

```
Daily Schedule → Get Weather → Extract Temperature → Temp > 30?
```

- **Daily Schedule** fires once every day at 00:00 (midnight).
- **Get Weather** queries the Open‑Meteo API for the current temperature.
- **Extract Temperature** pulls the first temperature value from the response (`hourly.temperature_2m[0]`) and stores it as `temperature`.
- **Temp > 30?** evaluates whether the extracted temperature exceeds 30 °C. The **true** branch can be extended with further actions (e.g., send a notification), while the **false** branch can remain empty or be used for alternative handling.

Feel free to adjust the latitude/longitude, the cron timing, or any other parameters to suit your needs.
